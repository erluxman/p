# Prompt Packs (Portable)

These are reusable prompt templates meant to be used across IDEs/agents.

## How to use

- Start sessions with:
  - `@project_constitution.md @workflow_protocol.md`
- For completion gates:
  - `@ai-dev/checklists/code_review_checklist.md`
  - `@ai-dev/checklists/security_checklist.md`
  - `@ai-dev/checklists/testing_checklist.md`
- Copy one template into chat and fill placeholders.

## Prompt packs

- `feature_spec.md`: spec template
- `plan.md`: planning template (files + risks)
- `tdd_bugfix.md`: repro test → fix → regression
- `refactor.md`: behavior-preserving refactor template
- `review.md`: structured code review request
- `release.md`: pre-ship checklist prompt
